###No-ip on vm
als root user werken op het systeem
```
sudo su
```
```
cd /usr/local/src/
wget http://www.no-ip.com/client/linux/noip-duc-linux.tar.gz
tar xf noip-duc-linux.tar.gz
cd noip-2.1.9-1/
make install
```
####daarna inloggen met no-ip.org account
